Grid-media System

The flexus grid-media system is not based in 12 columns. Instead of that, flexus has columns based in pixels which are breakpoints at the same time without media-queries. Let see this more deeper:

flex-1-1-500

Flex is the prefix to set a container.

The first "1" means grow. For example, if you set 500px in the last position and there is more space available in the father div, this "1" grows the container until there is not more free space.

if you set "grow" in "0" the container will stay in the measure that you set previously in "basis", in this case, 500px.

If you set "grow" in "2" (1 to 5) and besides the father container has free space, this child container will grow the double than its brothers containers. it's up to the number you set in "Grow".

The second "1" means shrink. if you set "0" the container won't be responsive, so for recomendation always set this value in "1".

This last property set the max-width of the container and the same time are breakpoint for itself. don't worry, it will be responsive in any device and resolution as long as shrink is set in 1.

You can set it since flex-0-1-100, -150, -200, -250 and so on until flex-0-1-1300.

Layout direction

By default, the direction of a flex container is set to row so the child elements display horizontally. Flex items

Main-axis alignments

The justify-content property aligns flex items along the main axis of the current line of the flex container. It helps distribute left free space in the flex container. So If "column" is not set, we can distribute flex items horizontally with the prefix "m-": m-start, m-center, m-end, m-around and m-between.

m-start

Flex items are set toward at the start (left) of the main axis (top, whether you are using column).

               
                  
            <div class="flex-body">
                     <div class="flex-main">
                              <div class="flex-1-1-400 m-start">
                                       <div>start</div>
                              </div>
                              <div class="flex-0-1-400">

                              </div>
                     </div>
            </div>
